Please read before create a scrim!

1. Currently, we only support Scrim for 5v5 tournament draft mode and summoner's rift.
2. You game detail will be recorded for our future feature "Reporting".
3. Tournament code will be provided to both Clan 1 hour before the scrim's Start Time.
4. You cannot change opponent after choosen one. You can remove the scrim to make a new one.
5. You can have up to 3 active (un-removed) scrims in the next 7 days. Please manage your time well
6. You can use the tournament code for repeated games during your scrim.
7. Do not use the tournament code for other purpose other than Scrim. You may get banned if the tournament code is being abused.
9. Need help with more information, please contact our discord staff.
