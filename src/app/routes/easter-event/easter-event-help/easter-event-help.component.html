Return to Event

This event is to celebrate Easter by using 4 champions (Ivern, Lulu, Riven, Fizz) and go through a series of adventures to complete the story.

There are two types of quest in this event.
  • Individual Quest : A individual quest is designed for granting score once the task is finish.
  • Group Quest (G): A group quest is needed to unlock the next stage. This quest is required by all players to complete. It does not give score, however event will continue to next stage if requirement are met.
In the event main page, you can submit the gameId of your game at the top right of the page. GameId is a unique ID that provide by Riot once you complete a riot game. You can see your gameId in your Match History in league of legend client. It usually at the top side of the match detail screen. If you have problem finding your gameId, Please contact our admin in discord.
  • Game must be played in either Blind Pick Mode or Draft Pick Mode or ARAM or Rank Solo or Rank Flex.
  • Game must use one of following champions: Ivern, Riven, Lulu, Fizz
  • Game must be played within the event period.
  • Game cannot be reused for same player.
  • Player cannot queue the game with friends from the other team and fed the states to complete the quest. if we found someone intentionally cheating the states, player involved maybe banned from the event.
  • However player can queue with friends within the same team.
  • Same GameId can be used by different player if those player are in that game.
  • Multiple quests can be completed by submitting one game ID.
